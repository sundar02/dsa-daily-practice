# 🧠 What is DSA?

**DSA = Data Structures + Algorithms**

### Data Structures

A **Data Structure** is a way of organizing and storing data so it can be accessed and modified efficiently.

Examples:

- Array
- Linked List
- Stack
- Queue
- Tree
- Graph
- Hash Table

### Algorithms

An **Algorithm** is a step-by-step procedure used to perform calculations, process data, or solve problems.

Examples:

- Binary Search
- Sorting Algorithms
- Graph Traversals
- Dynamic Programming

Together, **Data Structures and Algorithms** help developers write efficient and optimized programs.

# 📚 Complete DSA Topics

Below are the major topics in **Data Structures and Algorithms**.

---

## 1. Complexity Analysis

Understanding how efficient an algorithm is.

Topics:

- Time Complexity
- Space Complexity
- Big O Notation
- Big Theta
- Big Omega
- Amortized Analysis

---

## 2. Arrays

The simplest and most fundamental data structure.

Topics:

- Array traversal
- Prefix sums
- Sliding window
- Two pointers
- Kadane’s algorithm
- Matrix operations

---

## 3. Strings

Manipulation and processing of text data.

Topics:

- String matching
- Palindromes
- Anagrams
- Pattern searching
- String hashing

Advanced:

- KMP Algorithm
- Rabin-Karp Algorithm

---

## 4. Linked Lists

Types:

- Singly Linked List
- Doubly Linked List
- Circular Linked List

Topics:

- Insertion
- Deletion
- Reversal
- Cycle detection
- Merge lists

---

## 5. Stack

Principle:
**LIFO – Last In First Out**

Operations:

- Push
- Pop
- Peek

Applications:

- Valid Parentheses
- Expression evaluation
- Next Greater Element

---

## 6. Queue

Principle:
**FIFO – First In First Out**

Types:

- Simple Queue
- Circular Queue
- Deque
- Priority Queue

Applications:

- BFS traversal
- Scheduling systems

---

## 7. Hashing

Data Structures:

- HashMap
- HashSet

Topics:

- Hash functions
- Collision handling
- Frequency counting

Applications:

- Fast lookups
- Duplicate detection

---

## 8. Trees

A hierarchical data structure.

Concepts:

- Root
- Parent
- Child
- Leaf
- Height
- Depth

Types:

- Binary Tree
- Binary Search Tree (BST)
- AVL Tree
- Red-Black Tree
- Segment Tree
- Fenwick Tree

---

## 9. Tree Traversals

Depth First Search (DFS):

- Preorder
- Inorder
- Postorder

Breadth First Search (BFS):

- Level Order Traversal

---

## 10. Heaps / Priority Queue

Types:

- Min Heap
- Max Heap

Applications:

- Top K problems
- Scheduling
- Heap Sort

---

## 11. Graphs

Graphs represent relationships between objects.

Graph Types:

- Directed Graph
- Undirected Graph
- Weighted Graph
- Directed Acyclic Graph (DAG)

Representations:

- Adjacency List
- Adjacency Matrix

---

## 12. Graph Algorithms

Traversal:

- Depth First Search (DFS)
- Breadth First Search (BFS)

Advanced Algorithms:

- Topological Sort
- Dijkstra’s Algorithm
- Bellman-Ford Algorithm
- Floyd-Warshall Algorithm
- Minimum Spanning Tree

MST Algorithms:

- Kruskal
- Prim

---

## 13. Recursion

Recursion occurs when a function calls itself.

Key Concepts:

- Base case
- Recursive case
- Recursion tree
- Stack frames

---

## 14. Backtracking

Technique used to explore all possible solutions.

Applications:

- N-Queens
- Sudoku Solver
- Permutations
- Subsets
- Combination Sum

---

## 15. Dynamic Programming (DP)

An optimization technique used for problems with **overlapping subproblems**.

Types:

### 1D Dynamic Programming

Examples:

- Climbing Stairs
- House Robber
- Coin Change

### 2D Dynamic Programming

Examples:

- Longest Common Subsequence
- Edit Distance
- Unique Paths

---

## 16. Greedy Algorithms

Greedy algorithms make the **locally optimal choice** at each step.

Examples:

- Jump Game
- Activity Selection
- Gas Station Problem

---

## 17. Divide and Conquer

Divide a problem into smaller parts and combine results.

Examples:

- Merge Sort
- Quick Sort
- Binary Search

---

## 18. Bit Manipulation

Working directly with binary representations.

Operations:

- AND (&)
- OR (|)
- XOR (^)
- NOT (~)
- Left Shift (<<)
- Right Shift (>>)

Applications:

- Subsets
- Single number problems
- Bitmasking

---

## 19. Advanced Data Structures

More specialized structures used in advanced problems.

Examples:

- Trie
- Segment Tree
- Fenwick Tree (Binary Indexed Tree)
- Disjoint Set (Union Find)
- Sparse Table

---

## 20. Mathematical Algorithms

Important in competitive programming.

Topics:

- Prime numbers
- GCD / LCM

- Modular arithmetic
- Fast exponentiation
